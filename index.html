📱 카카오 개발자 앱 REST API 키를 입력해 주세요

처음 하시는 분:
1. developers.kakao.com 접속 → 로그인
2. 내 애플리케이션 → 애플리케이션 추가하기
3. 앱 이름 아무거나 입력 후 저장
4. 생성된 앱 클릭 → 앱 키 → REST API 키 복사

※ 아직 카카오 앱이 없으신 경우 위 안내대로 만들어 주세요 (2분 소요)

🐙 GitHub 토큰 입력

GitHub Personal Access Token 만들기:
1. github.com/settings/tokens/new 접속
2. Note: 상상우리 입력
3. Expiration: No expiration 선택
4. repo 체크박스 선택
5. Generate token → 토큰 복사 (ghp_로 시작)
